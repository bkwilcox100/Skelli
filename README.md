# Skelli (BETA)
A simple tool for quickly creating files

## Installation

#### NPM
    npm skelli -g

## Usage

#### Basic Format
    skelli [format] [name] [options...]

#### Creating a Text File
    skelli txt hello


## Templates Supported
+ HTML
+ CSS
+ Java
+ C++
+ Go
+ NPM Package
+ ReadMe Markdown File
+ Python
