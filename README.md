# Skelli (BETA)
A simple tool for quickly creating files

## Installation

#### NPM
    npm skelli -g

## Usage

#### Basic Format
    skelli [command] [name] [options...]

#### Creating a Text File
    skelli txt hello        
    // Outputs a single text file

    skelli txt hello -m 4   
    // Outputs 4 txt files (hello0.txt, hello1.txt...)
    // Up to 100 at a time
